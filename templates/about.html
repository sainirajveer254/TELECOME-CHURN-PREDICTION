📱 About Telecom Plan Recommender

🔗 LinkedIn | 🔗 Instagram

This intelligent web application helps users discover the best mobile recharge plans and broadband options based on their budget and usage needs.

💡 Project Overview

With numerous telecom and broadband plans available, it can be hard to find the one that fits your needs. This system simplifies the process by recommending the most suitable plans based on:

  • Price (₹)
  • Validity (days)
  • Daily Data (GB)
  • Preferred Plan Category (optional)
  • Broadband Speed (optional)

🧠 Technology Stack

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Python (Flask)
  • ML Model: XGBoost Classifier
  • Data: CSV files for telecom recharge plans and broadband options

🔍 How It Works

  1. User provides basic input (price, validity, etc.)
  2. ML model predicts the suitable plan category (if not selected)
  3. Plans are filtered and sorted by value
  4. Top 3–4 plans are displayed as recommendations
  5. Broadband recommendations are made based on speed preferences (optional)

🎯 Key Features

  • No input is mandatory – minimal input gives relevant results
  • Category-based prediction for personalized suggestions
  • Intuitive UI and responsive design
  • Plans sorted by value (price per GB)
  • New Feature: Broadband plan recommendations based on user’s speed and usage preferences

📈 Future Improvements

  • Live API integration for real-time telecom and broadband plans
  • User profile memory for better long-term recommendations
  • Signal strength and area-wise coverage info

📶 Network Speed Analysis

The app also includes a Network Speed Test feature, which allows you to test the download and upload speeds of your internet connection in real-time. Based on the results, the system will recommend whether switching to a better ISP or broadband provider might improve your internet experience.

  • The speed test checks your current download and upload speeds.
  • It compares your current ISP speed with the best available ISP speeds in your area.
  • Suggestions are made for a better broadband provider if a faster option is available.
